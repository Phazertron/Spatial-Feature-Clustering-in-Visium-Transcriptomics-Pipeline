# Contributing Guidelines

Thank you for your interest in contributing to the **Spatial Feature Clustering in Visium Transcriptomics** pipeline!  
This project aims to provide a clean, modular, and reproducible framework for multi-view gene clustering on 10x Genomics Visium data.

Contributions of all kinds are welcome — bug fixes, new modules, documentation improvements, or scientific enhancements.

---

## Project Structure Overview

Key directories:

- `src/` — Core pipeline modules (data loading, preprocessing, similarity, clustering, evaluation)
- `notebooks/` — Analysis workflow (01–06)
- `examples/` — Minimal sample configuration and output artifacts
- `scripts/` — Report generation and automation
- `data/` — Input datasets (ignored by Git)
- `results/` — Output artifacts (ignored by Git)
